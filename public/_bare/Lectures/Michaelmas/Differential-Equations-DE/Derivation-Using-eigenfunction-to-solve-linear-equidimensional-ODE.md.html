IMO this is a worse method than substitution.

Note that $x^k$ is an eigenfunction since $$\begin{align*} x \frac{d}{dx} x^k &= k x^k \\ x^2 \frac{d^2}{dx^2} x^k &= k(k-1)x^k \end{align*}$$ So we look for complimentary function of the form $y_c(x) = x^k$. Then $$\begin{align*} ak(k-1)x^k + bkx^k + cx^k &= 0 \\ ak^2 + (b-a)k + c &= 0 \end{align*}$$ which is our characteristic function.

If we have distinct roots $k_1, k_2$, then the general complimentary function is $$ y_c(x) = Ax^{k_1} + Bx^{k_2}. $$
