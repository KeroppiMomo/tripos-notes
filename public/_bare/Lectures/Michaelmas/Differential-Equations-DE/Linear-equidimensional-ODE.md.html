A linear, second-order equidimensional ODE has the form $$ ax^2 y'' + bxy' + cy = f(x) $$ where $a,b,c$ are constants.

Proposition (scaling property of solutions of homogenous linear equidimensional ODE) open_in_new

For a homogenous linear second-order equidimensional ODE $$ ax^2 y'' + bxy' + cy = 0, $$ if $y(x)$ is a solution, then $\tilde y(x) = y(\lambda x)$ is also a solution for some non-zero constant $\lambda$.

Proof

The solution has form $y(x) = x^k$ where the characteristic equation is $$ ak^2 + (b-a)k + c = 0. $$

  • If the roots $k_1 \neq k_2$, the solution is $$ y(x) = Ax^{k_1} + Bx^{k_2}. $$

  • If the roots $k_1=k_2 = k$, the solution is $$ y(x) = (A + B\ln x)x^k. $$

(Derivation using eigenfunction, derivation using substitution)
