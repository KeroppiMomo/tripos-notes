Definition (exponential function as infinite series) open_in_new

Define $$ \exp(x) = \sum_{n=0}^\infty \frac{x^n}{n!}.$$

Definition (exponential function as limit) open_in_new

Definition (exponential function as solution to f'=f) open_in_new

Alternatively define $\exp(x)$ as the unique solution of $$ \frac{df}{dx} = f\quad\text{with } f(0) = 1.$$

Proposition (adding exponents) open_in_new

Definition (Euler's number) open_in_new

Define $$ \exp(x) = e^x $$ where $$e = \exp(1) = \lim_{k\to\infty}\Big(1+\frac1k\Big)^k.$$

Definition (natural log) open_in_new

Define the natural log $\ln$ be the inverse of $\exp$.

Definition (power) open_in_new

Define for any $a,x\in\R$, $$ a^x = \exp(x\ln a). $$
