Definition (linear differential operator) open_in_new

An operator $\mathcal D$ is linear if for any functions $y_1(x), y_2(x)$ and constants $\alpha,\beta$, $$ \mathcal D(\alpha y_1 + \beta y_2) = \alpha\mathcal D y_1 + \beta\mathcal D y_2. $$

Definition (linearly independence of functions) open_in_new

A set of $N$ functions $\{f_i(x)\}$ is linearly dependent if $$ \sum_{i=1}^N c_i f_i(x) = 0 $$ for $N$ constants $c_i$ not all zero.

Otherwise the functions are linearly independent.

General form $$ a \frac{d^2y}{dx^2} + b\frac{dy}{dx} + cy = f(x). $$

Homogenous solution of second-order ODE with constant coefficient open_in_new

$$ a\frac{d^2y}{dx^2} + b\frac{dy}{dx} + cy = 0. $$ Solution has the form $y(x) = e^{\lambda x}$, so the characteristic equation is $$ a\lambda^2 + b\lambda + c = 0. $$

  • If the roots $\lambda_1 \neq \lambda_2$, then $$ y(x) = Ae^{\lambda_1 x} + Be^{\lambda_2 x}. $$

  • If the roots $\lambda_1 = \lambda_2 = \lambda$, then $$ y(x) = (A + Bx)e^{\lambda x}. $$
