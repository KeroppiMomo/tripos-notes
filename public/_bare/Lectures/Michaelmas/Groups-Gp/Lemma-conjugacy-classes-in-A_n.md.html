Let $\gamma\in A_n\subseteq S_n$

  1. If some odd element of $S_n$ commutes with $\gamma$, then $$\mrm{ccl}_{A_n}(\gamma) = \mrm{ccl}_{S_n}(\gamma).$$

  2. Otherwise i.e. if all elements of $S_n$ that commute with $\gamma$ are even, then the conjugacy class splits into two, $$\mrm{ccl}_{S_n}(\gamma) = \mrm{ccl}_{A_n}(\gamma) \union \mrm{ccl}_{S_n}(\tau\gamma\tau)$$ where $\tau$ is any transposition.
