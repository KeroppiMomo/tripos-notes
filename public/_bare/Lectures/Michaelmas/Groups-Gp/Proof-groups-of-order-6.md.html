By Cauchy's theorem, there exists $r,s\in G$ such that $o(r) = 3$ and $o(s) = 2$ since 2 and 3 are primes. Note that $s\not\in\gen r$ since no elements in $\gen r$ has order 2.

Now $$ |G : \gen r| = \frac{|G|}{|\gen r|} = \frac63 = 2, $$ so $\se{\gen r, s\gen r}$ partition $G$. At the same time, $\se{\gen r, \gen r s}$ also partition $G$, so $$ s\gen r = G\setminus \gen r = \gen r s.$$

This means that $sr = r^k s$ for some $k=0,1,2$.

  • For $i=0$, we have $sr=s$ so $r=e$, which does not have order 3, contradiction.

  • For $i=1$, we have $sr = rs$. This means $$ sr^2 = rsr = r^2 s $$ so $\gen r$ and $\gen s$ commute. Therefore, $$\begin{align*} G &\cong \gen r\times\gen s &&\text{by direct product theorem} \\ &\cong C_3\times C_2 \\ &\cong C_6 &&\text{by Chinese remainder theorem.} \end{align*}$$

  • For $i=2$, we have $sr = r\inv s$, which is the dihedral relation. We can list the elements of $G$, $$ G = \set{e, r, r^2, s, rs, r^2 s}, $$ and verify using the dihedral relation that the multiplication rules hold:

- - clearly $r^i r^j = r^{i+j}$; - $(r^i)(r^j s) = r^{i+j} s$; - $(r^i s)(r^j) = r^i r\inv s r^{j-1} = \ldots = r^i r^{-j} s = r^{i-j} s$; - $(r^i s)(r^j s) = r^i r^{-j} s s = r^{i-j}$,

which agrees with the multiplication table of the dihedral group. Therefore, $$ G\cong D_6. $$
