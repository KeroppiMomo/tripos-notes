• $(G,+,0)$ where $G = \mathbb Z,\mathbb Q,\mathbb R,\mathbb C$ is an abelian group.

  • $(\mathbb N, +, 0)$ is not a group because there is no inverse.

  • $(\mathbb Q, \times, 1)$ is not a group because 0 has no inverse.

  • $(\mathbb Q\setminus\{0\}, \times, 1)$ is an abelian group. Same for $\mathbb R$ and $\mathbb C$.
