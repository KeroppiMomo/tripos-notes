Example (all conjugacy classes in $A_4$) open_in_new

Refer to the conjugacy classes in $S_4$. Recall that a permutation is even iff it has an even number of even cycles.

All even elements of $S_4$ are

  • $e$;

  • $(2,2)$-cycles; and

  • 3-cycles.

Since $\pmat{1&2}$ commutes with $\pmat{1&2}\pmat{3&4}$, $$ \mrm{ccl}_{A_4}(\pmat{1&2}\pmat{3&4}) = \mrm{ccl}_{S_4}(\pmat{1&2}\pmat{3&4}). $$

For a 3-cycle $\sigma = \pmat{1&2&3}$, we know that $|C_{S_4}\pmat{1&2&3}| = 3$. We can list the elements of the centraliser by um guessing: $$ C_{S_4}\pmat{1&2&3} = \set{e, \pmat{1&2&3}, \pmat{3&2&1}}, $$ which are all even. Therefore $\mrm{ccl}_{S_4}\pmat{1&2&3}$ splits.

Example (all conjugacy classes in $A_5$) open_in_new

Using the conjugacy classes in $S_5$, we examine cycle types that give even permutations:

  • $e$ is boring.

  • $\pmat{1&2&3}$: we have odd $\pmat{4&5}\in C_{S_5}\pmat{1&2&3}$ so the conjugacy class does not split.

  • $\pmat{1&2}\pmat{3&4}$: we have odd $\pmat{1&2}\in C_{S_5}\big(\pmat{1&2}\pmat{3&4}\big)$ so the conjugacy class does not split.

  • $\pmat{1&2&3&4&5}$: note that $|C_{S_5}\pmat{1&2&3&4&5}| = 5$ and also $C_{S_5}(\pmat{1&2&3&4&5}) \supseteq \gen{\pmat{1&2&3&4&5}}$ has size 5. Therefore $C_{S_5}(\pmat{1&2&3&4&5}) = \gen{\pmat{1&2&3&4&5}} \subseteq A_5$.

    Hence the conjugacy class splits. The other one has typical element $$ \pmat{1&2}\pmat{1&2&3&4&5}\pmat{1&2} = \pmat{2&1&3&4&5}. $$
