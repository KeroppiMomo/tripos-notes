Part 1: Defining an action

The clever bit is to consider this set $$ X = \sb{(g_1,g_2,\ldots, g_p)}{g_i\in G\ \forall i, \ g_1g_2\ldots g_p = e}. $$

Define an action $C_p = \langle t\rangle$ on $X$, $$ t^k (g_1, g_2, \ldots, g_p) = (g_{k+1}, g_{k+2},\ldots, g_p, g_1, g_2, \ldots, g_k) $$ which is to shift the tuple by $k$ spots. We need to check whether this is an action.

Firstly it's easy to see that $e=t^0$ acts trivially and that $t^k t^l (g_1,\ldots, g_p) = t^{k+l} (g_1,\ldots, g_p)$.

Next we have to check $t^k (g_1,\ldots,g_p)\in X$. Suppose $g_1g_2\ldots g_p = e$. This means $$ (g_1g_2\ldots g_k)(g_{k+1}g_{k+2}\ldots g_p) = e, $$ so $$ t^k(g_1\ldots g_p) = (g_{k+1}g_{k+2}\ldots g_p)(g_1g_2\ldots g_k) = e. $$

Hence we have an action of $C_p$ on $X$.

Part 2: Computing $|X|$

For every $(g_1,g_2,\ldots,g_p) \in X$, notice the restriction $$ g_1 g_2\ldots g_p = e \iff g_p = (g_1g_2\ldots g_{p-1})^{-1}. $$ Therefore you can pick any element of $G$ for $g_1, g_2, \ldots g_{p-1}$ and then $g_p$ is fixed. Hence, $$ |X| = |G|^{p-1}. $$

Another method to approach this is to note that the action of $C_p$ partitions $X$ into $m$ orbits for some $m$, $$ X = C_p x_1 \cup C_p x_2 \cup \ldots \cup C_p x_m. $$ By orbit-stabiliser, for all $j$, $$ |C_p x_j|\ \big|\ |C_p|=p $$ so $$|C_p x_j| = 1 \text{ or } p.$$

Let $l$ be the number of orbits of size 1. Then $m-l$ orbits have size $p$. Then $$ |X| = l(1) + (m-l)(p) = l + (m-l)p. $$

Combining two results abut $|X|$, $$ |G|^{p-1} = l + (m-l)p. $$

Using the hypothesis that $p\ \big|\ |G|$, we have $p\ | \ l$ and therefore $$ l = 0 \text{ or } l\geq p. $$

Part 3: Orbits of $C_p$

If $|C_p x| = 1$, it means that if $x = (g_1, g_2,\ldots, g_p)$, $$ t^k x = x \quad\forall k $$ so $$ x = (g, g, \ldots, g)\quad\text{for some } g\in G.$$

Since $\underbrace{e\cdot e\cdot\ldots\cdot e}_{p\text{ times}} = e$, $$ (e, e, \ldots, e) \in X $$ provides an orbit of size 1.

Therefore $l\neq 0$, and by above, we have $l\geq p\geq 2$ such orbits.

Thus there exists $g\neq e$ such that $(g,g,\ldots,g)\in X$ with $g^p = e$, as required.
