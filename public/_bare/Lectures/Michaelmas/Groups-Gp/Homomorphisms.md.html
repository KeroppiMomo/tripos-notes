Definition (homomorphism) open_in_new

A map between groups $$ \phi : G\to H $$ is a homomorphism if $$ \phi(g\cdot g') = \phi(g)\cdot\phi(g')\quad \forall g,g'\in G. $$

Lemma (homomorphism sends identity to identity) open_in_new

If $\phi: G\to H$ is a homomorphism, then $$\phi(e_G) = e_H.$$

Proof

Lemma (homomorphism preserves inverses) open_in_new

If $\phi: G\to H$ is a homomorphism, then $$ \phi(g^{-1}) = (\phi(g))^{-1} \quad\forall g\in G. $$

Proof

Lemma (composition of homomorphisms) open_in_new

If $G\xrightarrow{\phi} H\xrightarrow{\psi} K$ are homomorphisms, so is their composition $\psi\circ\phi$.

Proof

Proposition (image on generating set determines homomorphism) open_in_new

If $\phi: G \to H$ be a homomorphism and $G = \gen X$, then $\phi$ is uniquely determined by the image $\phi(X)$. (Example sheet 1, Q9)

Proof

Examples of homomorphisms open_in_new

Trivial homomorphism open_in_new

For any group $G,H$, a map $$\begin{align*} \phi: G&\to H \\ g&\mapsto e_H \end{align*}$$ is the trivial homomorphism.

Inclusion homomorphism open_in_new

If $H\leq G$, then the map $$\begin{align*} i_H : H&\to G \\ h&\mapsto h \end{align*}$$ is the inclusion homomorphism.

Homomorphisms between roots of unity open_in_new

For $C_n = \{z\in\mathbb C \mid z^n = 1\}$, if $n\mid m$, then $$\begin{align*} \phi: C_m &\to C_n \\ z &\mapsto z^{m/n} \end{align*}$$ is a homomorphism. Proof

Determinant as homomorphism open_in_new

The map $$\begin{align*} \mathrm{GL_n}(\mathbb R) &\to \R_\times = (\mathbb R\setminus \{0\},\times,1) \\ A &\mapsto \det A \end{align*}$$ is a homomorphism since $$ \det(AB) = \det(A)\det(B)$$

Definition (isomorphism) open_in_new

If a homomorphism $\phi: G\to H$ is also a bijection, then $\phi$ is an isomorphism, we say that $G$ is isomorphic to $H$, denoted by $G\cong H$.

Lemma (inverse of isomorphism is isomorphism) open_in_new

If $\phi: G\to H$ is an isomorphism, so is $\phi^{-1}$.

Proof

Image and kernel open_in_new

Definition (image and kernel of homomorphism) open_in_new

Let $\phi: G\to H$ be a homomorphism. Its image is defined as $$ \mathrm{im}(\phi) = \{\phi(g)\in H\mid g\in G\}. $$ Its kernel is defined as $$ \mathrm{ker}(\phi) = \{g\in G\mid \phi(g) = e_H\}. $$

Proposition (image and kernel are subgroups) open_in_new

If $\phi: G\to H$ is a homomorphism, then

  1. $\mathrm{im}\,\phi \leq H$;

  2. $\mathrm{ker}\,\phi \leq G$.

Proof

Proposition (isomorphism, image and kernel) open_in_new

Let $\phi: G\to H$ be a homomorphism.

  1. $\phi$ is surjective iff $\im\phi = H$.

  2. $\phi$ is injective iff $\ker\phi = 1_G = \{e_G\}$ is the trivial subgroup of $G$.

  3. $\phi$ is bijective iff $\im\phi = H$ and $\ker\phi = 1_G$.

Proof

Definition (cyclic group) open_in_new

A group $G$ is a cyclic if there is some $g\in G$ such that $$ G = \langle g\rangle = \sb{g^k}{k\in\Z}.$$ $g$ is called the generator of $G$.

Theorem (isomorphism of cyclic groups) open_in_new

If $G$ is cyclic, then either

  • $G\cong C_n$ for some $n\in\N_{> 0}$; or

  • $G\cong\Z$ (sometimes written as $C_\infty$).

Definition (order of group element) open_in_new

Let $G$ be a group. For any $g\in G$, let $$ \langle g\rangle = \sb{g^k}{k\in\Z} \leq G. $$ Since $\langle g\rangle$ is cyclic, $$ \langle g\rangle \cong C_n $$ for some $n \in \N_{>0}\cup\se\infty$. $n$ is called the order of $g$, denoted $o(g)$.

Proposition (group with all non-trivial elements of order 2 is abelian) open_in_new

For a group $G$, if all elements other than the identity have order 2, then $G$ is abelian.
