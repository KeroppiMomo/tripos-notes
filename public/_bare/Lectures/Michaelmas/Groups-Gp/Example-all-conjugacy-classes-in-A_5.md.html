Using the conjugacy classes in $S_5$, we examine cycle types that give even permutations:

  • $e$ is boring.

  • $\pmat{1&2&3}$: we have odd $\pmat{4&5}\in C_{S_5}\pmat{1&2&3}$ so the conjugacy class does not split.

  • $\pmat{1&2}\pmat{3&4}$: we have odd $\pmat{1&2}\in C_{S_5}\big(\pmat{1&2}\pmat{3&4}\big)$ so the conjugacy class does not split.

  • $\pmat{1&2&3&4&5}$: note that $|C_{S_5}\pmat{1&2&3&4&5}| = 5$ and also $C_{S_5}(\pmat{1&2&3&4&5}) \supseteq \gen{\pmat{1&2&3&4&5}}$ has size 5. Therefore $C_{S_5}(\pmat{1&2&3&4&5}) = \gen{\pmat{1&2&3&4&5}} \subseteq A_5$.

    Hence the conjugacy class splits. The other one has typical element $$ \pmat{1&2}\pmat{1&2&3&4&5}\pmat{1&2} = \pmat{2&1&3&4&5}. $$
