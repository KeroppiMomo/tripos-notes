Using orbit-stabiliser, we have $$\begin{align*} |S_n| &= |\mrm{ccl}_{S_n}(\gamma)||C_{S_n}(\gamma)|, \\ |A_n| &= |\mrm{ccl}_{A_n}(\gamma)||C_{A_n}(\gamma)|. \end{align*}$$ Also since $|S_n|/|A_n| = |S_n:A_n| = 2$, we get the relation $$ |\mrm{ccl}_{S_n}(\gamma)| = \frac{2|C_{A_n}(\gamma)|}{|C_{S_n}(\gamma)|} |\mrm{ccl}_{A_n}(\gamma)|. $$

Now $C_{A_n}(\gamma) = C_{S_n}(\gamma) \inter A_n$, so $$ |C_{S_n}(\gamma) : C_{A_n}(\gamma)| \leq |S_n:A_n| = 2. $$

  1. If there exists an odd element of $S_n$ that commute with $\gamma$, then $C_{S_n}(\gamma) \neq C_{A_n}(\gamma)$, so $|C_{S_n}(\gamma):C_{A_n}(\gamma)| = 2$. Hence $$ |\mrm{ccl}_{S_n}(\gamma)| = \frac{2}{2}|\mrm{ccl}_{A_n}(\gamma)| = |\mrm{ccl}_{A_n}(\gamma)|. $$ Since $\mrm{ccl}_{A_n}(\gamma) \subseteq \mrm{ccl}_{S_n}(\gamma)$, $$ \mrm{ccl}_{A_n}(\gamma) = \mrm{ccl}_{S_n}(\gamma). $$

  2. If all elements in $S_n$ that commute with $\gamma$ are even, then $C_{S_n}(\gamma) = C_{A_n}(\gamma)$, so $$ |\mrm{ccl}_{S_n}(\gamma)| = 2|\mrm{ccl}_{A_n}(\gamma)|, $$ i.e. $\mrm{ccl}_{S_n}(\gamma)$ is split into two.

    Then consider any transposition $\tau\in S_n$. Note that $\tau\gamma\tau = \tau\gamma\tau\inv \in \mrm{ccl}_{S_n}(\gamma)$. Now we have to prove that it is $\not\in \mrm{ccl}_{A_n}(\gamma)$.

    Suppose $\tau\gamma\tau\inv \in \mrm{ccl}_{A_n}(\gamma)$, which means that there exists $\alpha\in A_n$ such that $$\tau\gamma\tau\inv = \alpha\gamma\alpha\inv.$$ But then $$ (\alpha\inv\tau)\gamma = \gamma(\alpha\inv\tau) $$ which contradicts with no odd elements in $S_n$ commuting with $\gamma$.
