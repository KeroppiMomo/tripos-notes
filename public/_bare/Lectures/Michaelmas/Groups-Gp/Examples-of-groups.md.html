Trivial group open_in_new

For $G=\{e\}$ with $e\cdot e=e$, $(G,\cdot,e)$ is a group called the trivial group.

Groups from arithmetic open_in_new

  • $(G,+,0)$ where $G = \mathbb Z,\mathbb Q,\mathbb R,\mathbb C$ is an abelian group.

  • $(\mathbb N, +, 0)$ is not a group because there is no inverse.

  • $(\mathbb Q, \times, 1)$ is not a group because 0 has no inverse.

  • $(\mathbb Q\setminus\{0\}, \times, 1)$ is an abelian group. Same for $\mathbb R$ and $\mathbb C$.

Examples of finite groups open_in_new

Definition (order of group, finite group) open_in_new

The order of a group $(G,\cdot,e)$ is the number of elements of $G$, denoted by $|G|$.

If $|G|$ is not infinite then $(G,\cdot,e)$ is a finite group.

For all $n=1,2,3,\ldots$,

  • let $C_n = \{z\in\mathbb C : z^n = 1\}$. Then $(C_n,\times,1)$ is an abelian group of order $n$.

  • Let $\mathbb Z_n = \{0,1,2,\ldots, n-1\}$. Let $a+b$ be the remainder of $a+b$ divided by $n$ for any $a,b\in\mathbb Z_n$. Then $(\mathbb Z_n, +, 0)$ is a group. Proof

Definition (general linear group) open_in_new

Define the general linear group be $$\mathrm{GL}_n(\R) = \sb{A\in M_n(\R)}{\det A \neq 0}.$$ Same for $\C$. This is a group under multiplication.

This is an example of an infinite non-abelian group.

Symmetric groups open_in_new

Let $X$ be a set.

Definition (permutation) open_in_new

A permutation is a bijective function $f: X\to X$. This means that $f$ has an inverse $f^{-1}:X\to X$ where $$ f\circ f^{-1} = Id_X = f^{-1}\circ f. $$

Definition (symmetric groups) open_in_new

$\mathrm{Sym}(X)$ is defined to be the set of all permutations of $X$.
