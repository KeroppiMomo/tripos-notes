Normal subgroups open_in_new

Definition (normal subgroup) open_in_new

A subgroup $H\leq G$ is called normal if $$ ghg\inv \in H \quad \forall h\in H, g\in G. $$ We denote this as $H\lhd G$.

Example of normal subgroups open_in_new

  • For all group $G$, $1\lhd G$ and $G\lhd G$.

  • If $G$ is abelian, then any $H\leq G$ is normal.

  • $\gen r\lhd D_6$ since $$\begin{align*} srs\inv = r\inv ss\inv = r^2 &\in \gen r \\ sr^2s\inv =r^{-2} ss\inv = r &\in \gen r \end{align*}$$ etc, but $\gen s$ is not normal since $$ rsr\inv = sr^{-2} = sr \not\in \gen s. $$

Proposition (kernel is a normal subgroup) open_in_new

Suppose $\phi: G\to K$ is a homomorphism. Then $\ker\phi\lhd G$.

Proof

Lemma (left and right cosets equal iff normal subgroup) open_in_new

Suppose $H\leq G$. Then $H\lhd G$ iff $gH = Hg$ for all $g\in G$.

Proof

Proposition (normal subgroup is a union of conjugacy classes) open_in_new

If $H \lhd G$, then $H$ is the union of $\ccl_{G}(\gamma)$ for some elements $\gamma\in G$.

Theorem (quotient groups) open_in_new

If $H\lhd G$, then the set of (left) cosets $G/H$ is a group with operation $$ (g_1 H)(g_2 H) = (g_1 g_2) H.$$

Proof

Definition (quotient group) open_in_new

If $H\lhd G$, then $G/H$ is called the quotient of $G$ by $H$.

Examples of quotients open_in_new

  • $G/1\isom G$ and $G/G \isom 1$.

  • Since $\Z$ is abelian, any subgroup $n\Z \lhd \Z$. Also the quotient $\Z/n\Z$ is generated by $1+n\Z$, so we have $\Z/n\Z \isom C_n$.

  • Proposition (subgroups of index 2) open_in_new

    Suppose $H\leq G$. If $|G:H| = 2$, then $H\lhd G$ and $G/H \isom C_2$.

    Proof (subgroups of index 2) open_in_new

    If $|G:H| = 2$, then for any $g\in G\setminus H$, $$ gH = G \setminus H = Hg $$ so $H\lhd G$. Also since $|G/H|=2$, we know that $G/H \isom C_2$.

  • $C_n = \gen r\leq D_{2n}$ and $|D_{2n} : \gen r| = 2$, so $\gen r\lhd D_{2n}$ and $D_{2n} / C_{2n} \isom C_2$.

  • Quotients are not subgroups!

  • Also you cannot "multiply" quotients, e.g. $C_4 / C_2 \isom C_2$ and $K_4/C_2\isom C_2$ but $C_4\not\isom K_4$.

Theorem (isomorphism theorem) open_in_new

If $\phi: G\to H$ is a homomorphism, then $$ \ker\phi \lhd G \quad \text{and} \quad G / \ker\phi \isom \im\phi. $$

Proof

Examples using isomorphism theorem open_in_new

  • $$\begin{align*} \phi: \Z&\to C_n \\ k&\mapsto e^{2\pi ik/n} \end{align*}$$ is a homomorphism with $\ker\phi = n\Z$ and $\im\phi = C_n$, so we have $\Z/n\Z \isom C_n$.

  • $$\begin{align*} \phi: \R&\to\C_\times \\ t&\mapsto e^{2\pi it} \end{align*}$$ is a homomorphism with $\ker\phi = \Z$ and $\im\phi = \sb{z\in\C}{|z| = 1} \equiv \mathbb S^1$, so we have $\R/\Z = \mathbb S^1$.

Simple groups open_in_new

Definition (simple group) open_in_new

A group $G$ is simple if the only normal subgroups are $1$ and $G$.

Proposition (homomorphism from simple group) open_in_new

If $\phi: G\to H$ is a homomorphism and $G$ is a simple group, then $\phi$ is either trivial or injective.

Proof

Proposition (C_p is simple) open_in_new

For any prime $p$, $C_p$ is a simple group.

Proof
