We want to shift the origin by some amount $\v a$ to eliminate the linear term, $$ \v x \leftarrow \v x + \v a. $$

Substituting, $$\begin{align*} 0 &= (\v x + \v a)^T D(\v x + \v a) + \v b^T (\v x + \v a) + c \\ &= \v x^T D \v x + \v a^T D \v x + \v x^T D\v a + \v a^T D\v a + \v b^T \v x + \v b^T \v a + c \\ &= \v x^T D \v x + \ub{(\v a^T D\v x + \v x^T D\v a + b^T\v x)}_{= 0} + \ub{(\v a^T D\v a + \v b^T\v a + c)}_\text{const.} \\ \end{align*}$$ We want the middle term to vanish, so we set $$\begin{align*} 0 &= (D^T\v a)^T \v x + \v x^T (D\v a) + \v b^T \v x\\ &= (D\v a)^T \v x + \v x^T (D\v a) + \v b^T \v x && \text{$D$ is diagonal} \\ &= (D\v a)^T \v x + (D\v a)^T \v x + \v b^T \v x && \text{transpose of scalar}\\ &= (2D\v a + \v b)^T \v x \end{align*}$$ so set $$ \v a = -\frac12 D\inv \v b. $$
