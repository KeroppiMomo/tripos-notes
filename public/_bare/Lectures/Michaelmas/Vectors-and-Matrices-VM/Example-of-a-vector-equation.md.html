Solve $$ \vec x - (\vec x \times \vec a)\times\vec b = \vec c $$ for $\vec x$.

Using vector triple product, $$\begin{align*} \vec c &= \vec x - (\vec x\cdot\vec b)\vec a + (\vec a\cdot\vec b)\vec x \\ &= (1+\vec a\cdot\vec b)\vec x - (\vec x\cdot\vec b)\vec a \\ \vec c\cdot\vec b &= (1+\vec a\cdot\vec b)(\vec x\cdot\vec b) - (\vec x\cdot\vec b)(\vec a\cdot b) \\ &= \vec x\cdot\vec b. \end{align*}$$

Putting this back, $$\begin{align*} \vec c &= (1+\vec a\cdot\vec b)\vec x - (\vec c\cdot\vec b)\vec a \\ (1+\vec a\cdot\vec b)\vec x &= \vec c + (\vec c\cdot\vec b)\vec a. \end{align*}$$

  • If $\vec a\cdot\vec b\neq 1$, then we have a single solution $$ \vec x = \frac{\vec c+(\vec c\cdot\vec b)\vec a}{1+\vec a\cdot\vec b}. $$

  • If $\vec a\cdot\vec b=1$, then $$\begin{align*} \vec c + (\vec c\cdot\vec b)\vec a &= \vec 0 \\ \vec c &= -(\vec c\cdot\vec b)\vec a. \end{align*}$$ -

    • If $\vec c\neq -(\vec c\cdot\vec b)\vec a$, then we have contradiction so no solution.

    • If $\vec c=-(\vec c\cdot\vec b)\vec a$, then our solution is $\vec x\cdot\vec b = \vec c\cdot b$, a plane of solutions.
