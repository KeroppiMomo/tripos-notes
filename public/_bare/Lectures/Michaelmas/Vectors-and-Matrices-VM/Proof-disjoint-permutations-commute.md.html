Let $x=1,2,\ldots,n$.

If $\nu(x)\neq x$, then $\mu(x) = x$. Also since $\nu\big(\nu(x)\big)\neq \nu(x)$ by injectivity, $\mu\big(\nu(x)\big) = \nu(x)$. Therefore $$ \mu\circ\nu(x) = \nu(x) =\nu\circ\mu(x). $$

If $\mu(x) \neq x$, then $\nu(x)=x$. By the same logic, we have $\mu\circ\nu(x) = \nu\circ\mu(x)$.

Finally if $\mu(x) = \nu(x) = x$, then trivially $\mu\circ\nu(x) = \nu\circ\mu(x) = x$.
