The distribution function of a random variable $X$ is the function $F_X: \R \to [0,1]$ given by $$ F_X(x) = \prob(X\leq x). $$

For a random variable $X$ on $\R^n$, the distribution function of $X$ (also called the joint distribution function $X_1, \ldots, X_n$) is defined as $F_X: \R^n \to [0,1]$ by $$ F_X(x_1, x_2, \ldots, x_n) = \prob(X_1\leq x_1,\ \ldots,\ X_n\leq x_n). $$
