Given a non-decreasing function $f:\set{1,2,\ldots,k}\to\set{1,2,\ldots,n}$, we can define an increasing function $$\begin{align*} g:&&\set{1,2,\ldots,k}&\to\set{1,2,\ldots,n+k-1} \\ && i&\mapsto f(i) + i - 1 \end{align*}$$ and we obtain all such increasing functions in this way.

So the number is $\pmat{n+k-1 \\ k}$.
