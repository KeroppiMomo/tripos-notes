Since $q_n\to q$, taking the limit in $q_{n+1}=F(q_n)$ gives $q=F(q)$.

$F(t)$ is continuous and non-decreasing on $[0,1]$ with $F(1)=1$. Let $t$ for the smallest non-negative solution of $t=F(t)$. Then $q_0=0\leq t$. Suppose inductively $q_n \leq t$, then $q_{n+1}=F(q_n)\leq F(t)=t$ since $F$ is non-decreasing, so the induction proceeds. Hence $q\leq t$, but since $q=F(q)$, we get $q=t$ as required.

Consider first the case $\mu\equiv\expect(X_1)>1$. Note that $F'(t)\to\mu$ as $t\to 1$ where $t\in[0,1)$, so $F'(t)>1$ for all $t$ sufficiently close to $1$, say $t>T$. Since $F(1)=1$, by the mean value theorem, $F(t)<t$ for $t>T$. But $F(0)=\prob(X_1=0) > 0$, so by the intermediate value theorem, there exists $t\in[0,1)$ such that $F(t)=t$, so $q<1$.

Next consider the case $\mu\leq 1$ with $$\prob(X_1\leq 1)=1.$$ Then $F(t) = 1-\mu+\mu t$. Since $\prob(X_1=1)<1$ is assumed, $\mu\neq1$, so the only solution to $F(t)=t$ is $t=1$. Hence $q=1$.

Finally consider the case $\mu\leq 1$ with $\prob(X_1\geq 2)>0$. Then $$ F'(t) = \fsum k1\infty kt^{k-1} \prob(X_1=k) $$ is strictly increasing. Now $F'(t)\to\mu\leq 1$ as $t\to1$, so $F'(t)<1$ for all $t\in[0,1)$. Hence by the mean value theorem, $F(t)>t$ for all $t\in[0,1)$ and so $q=1$.
