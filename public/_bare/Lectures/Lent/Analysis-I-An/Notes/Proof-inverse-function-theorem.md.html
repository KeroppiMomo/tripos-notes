• $f$ is injective: if $x\neq y$, then WLOG let $a\leq x<y\leq b$. Then $f(x) < f(y)$ so $f(x)\neq f(y)$.

  • $f$ is maps into $[c,d]$: for all $x$ where $a\leq x\leq b$, monotonicity implies $c = f(a) \leq f(x) \leq f(b) = d$, so indeed $f: [a,b]\to[c,d]$.

  • $f$ is surjective: for all $\eta\in[c,d]$, by the intermediate value theorem, there exists $x\in[a,b]$ with $f(x) = \eta$.

This proves that $f$ is bijective.

  • $f\inv$ is continuous: first consider $z\in(c,d)$ and let $y=f\inv(z)\in(a,b)$. For all $\epsilon>0$ that is sufficiently small that $a\leq y-\epsilon < y < y+\epsilon\leq b$, monotonicity implies $$ c = f(a) \leq f(y-\epsilon) < f(y) < f(y+\epsilon) \leq f(b) = d. $$ Pick $\delta > 0$ such that $(z-\delta, z+\delta)\subset \big(f(y-\epsilon), f(y+\epsilon)\big)$.

    Now if for some $z'\in[c,d]$ where $|z'-z| < \delta$, then since $f: (y-\epsilon, y+\epsilon) \to \big(f(y-\epsilon), f(y+\epsilon)\big)$ is a bijection, $$z'\in\big(f(y-\epsilon), f(y+\epsilon)\big) \implies f\inv(z') \in (y-\epsilon, y+\epsilon) $$ so $|f\inv(z') - f\inv(z)| < \epsilon$.

    If $z=c$ or $d$, a similar argument works. Hence $f\inv$ is continuous.

  • $f\inv$ is strictly increasing, suppose not. Then there exists $c\leq z_1 < z_2\leq d$ with $f\inv(z_1) \geq f\inv(z_2)$. But $f$ is strictly increasing, so $$ f\big(f\inv(z_1)\big) \geq f\big(f\inv(z_2)\big) \implies z_1 \geq z_2, $$ which is a contradiction.
