Lemma (Rolle's theorem) open_in_new

Let $f: [a,b]\to\R$ be continuous on $[a,b]$ and differentiable at $(a,b)$.

If $f(a)=f(b)$, then there exists $c\in(a,b)$ such that $f'(c)=0$.

Proof

Theorem (mean value theorem) open_in_new

Let $f: [a,b]\to\R$ be continuous on $[a,b]$ and differentiable on $(a,b)$.

Then there exists $c\in(a,b)$ such that $$ f(b) - f(a) = (b-a)f'(c). $$

Proof

Equivalently, if $f$ is differentiable on an open interval $I$ containing $a$, then for all $h\in\R$ where $a+h\in I$, there exists $\theta\in(0,1)$ such that $$ f(a+h) = f(a) + hf'(a+\theta h). $$

Corollary (derivative and monotonicity) open_in_new

Let $f: [a,b]\to\R$ be a continuous function which is differentiable on $(a,b)$. Then

  1. if $f'(x)>0$ for all $x\in(a,b)$, then $f$ is strictly increasing;

  2. if $f'(x)\geq 0$ for all $x\in(a,b)$, then $f$ is increasing;

  3. if $f'(x) = 0$ for all $x\in(a,b)$, then $f$ is constant.

Also if $f: \C\to\C$ with $f'(z)=0$ for all $z\in\C$, then $f$ is constant.

Theorem (Cauchy's mean value theorem) open_in_new
