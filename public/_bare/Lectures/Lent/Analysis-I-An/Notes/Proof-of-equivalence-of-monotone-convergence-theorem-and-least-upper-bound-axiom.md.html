LUB => MCT

For an increasing sequence $(a_n)$, let $$ A = \{a_1, a_2, a_3, \ldots\}. $$ Since $(a_n)$ is bounded above, $A$ is also bounded above. Hence by LUB, let $a = \sup A$.

To show that $a_n \to a$ as $n\to\infty$, let $\epsilon > 0$. Since $a$ is a supremum, there exists $a_N\in A$ where $a_N > a-\epsilon$. Then as $(a_n)$ is increasing, $a_n \geq a_N > a-\epsilon$. Also using $a_n \leq a$, we obtain $$ |a_n - a| < \epsilon\quad\forall n\geq N. $$ Hence $a_n\to a$.

MCT => LUB

Suppose $X\neq\emptyset$ is bounded above. We construct two sequences $(x_n)_{n\in\N}$ and $(y_n)_{n\in\N}$ as follows:

  • Step 1: let $x_1 \in X$ as $X$ is non-empty, and let $y_1$ be any upper bound of $X$.

  • Step $n$: let $z_n=(x_{n-1}+y_{n-1})/2$. We have two cases:

    • If $z_n$ is an upper bound of $X$, then set $x_n = x_{n-1}\in X$ and $y_n = z_n$.

    • Otherwise, pick $x_n\in X$ where $x_n > z_n$, and let $y_n = y_{n-1}$.

    In both cases,

    1. $x_{n-1}\leq x_n\in X$;

    2. $y_n$ is an upper bound of $X$ and $y_n \leq y_{n-1}$; and

    3. $y_n - x_n \leq (y_{n-1} - x_{n-1})/2$.

Now from (1), $(x_n)_{n\in\N}$ is an increasing sequence bounded above by any $y$, so by MCT, it converges to some value $a$.

Also from (2), $(y_n)_{n\in\N}$ is a decreasing sequence bounded below by any $x$, so it converges to some value $a'$.

From (3), since $y_n - x_n$ is squeezed between 0 and $(y_1-x_1)/2^{n-1}$, it has limit 0, so $a' = a$.

To show that $a$ is an upper bound of $X$, $y_n\to a$ means that for all $\epsilon>0$, there exists $N$ such that $|y_N - a| < \epsilon$. Since $y_N$ is an upper bound, for any $x\in X$, $$ x \leq y_N < a+\epsilon. $$ As $\epsilon$ can be arbitrarily small, $x\leq a$. Hence $a$ is an upper bound of $X$.

To show that $a$ is the least upper bound, note that $x_n\to a$ means that for all $\epsilon>0$, there exists $N$ where $x_N > a-\epsilon$.

Thus $\sup X = a$ exists.
