Since $n_{j+1} > n_j$, we have $n_{j+1} \geq n_j + 1$, so by induction, $n_j \geq j$.

Since $a_n\to a$, for all $\epsilon > 0$, there exists $N(\epsilon)$ such that $|a_n - a| < \epsilon$ if $n\geq N(\epsilon)$. Hence for all $j\geq N(\epsilon)$, as $n_j \geq j\geq N(\epsilon)$, we have $|a_{n_j} - a| < \epsilon$ as required.
