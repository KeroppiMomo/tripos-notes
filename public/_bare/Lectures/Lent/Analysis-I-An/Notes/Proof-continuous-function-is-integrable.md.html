(Since $f$ is continuous on a closed interval, by the extreme value theorem, $f$ is bounded, so it makes sense to talk about integrability.)

We'll prove the contrapositive, i.e. if $f$ is not integrable, then $f$ is not continuous.

Suppose $f$ is not integrable. By Cauchy criterion, there exists $\epsilon>0$ such that for all dissections $\mathcal{D}$, $$ \begin{align*} U(f,\mathcal{D}) - L(f,\mathcal{D}) > \epsilon. \end{align*} $$

For all dissections $\mathcal{D}$, $$ \begin{align*} U(f,\mathcal{D}) - L(f,\mathcal{D}) = \sum_{j=1}^{n} (x_j - x_{j-1})\left(\sup_{x\in[x_{j-1}, x_j]} f(x) - \inf_{x\in[x_{j-1}, x_j]} f(x)\right) \end{align*} $$ there exists some $j$ with $$ \begin{align*} \sup_{x\in[x_{j-1}, x_j]} f(x) - \inf_{x\in[x_{j-1}, x_j]} f(x) > \frac{\epsilon}{b-a} \end{align*} $$ because otherwise $U(f,\mathcal{D})-L(f,\mathcal{D}) \leq (\epsilon/(b-a))\sum_{j=1}^{n} (x_j-x_{j-1}) = \epsilon$.

Using definition of supermum and infimum, we can find $y,z\in[x_{j-1}, x_j]$ with $f(y) - f(z) > \epsilon/(b-a)$.

Apply the argument above to the uniform dissection $\mathcal{D}_n$ to obtain sequences $y_n, z_n$ where $|y_n-z_n| < (b-a)/n$ with $$ \begin{align*} f(y_n) - f(z_n) > \frac{\epsilon}{b-a}. \end{align*} $$

$(y_n)$ is a bounded sequence, so by Bolzano-Weierstrass, we can take a converging subsequence $y_{n_k} \to \eta$ for some $\eta\in[a,b]$. Since $|y_{n_k} - z_{n_k}| < (b-a)/n_k \leq (b-a)/k$, we also have $z_{n_k} \to \eta$.

But $f(y_{n_k}) - f(z_{n_k}) > \epsilon/(b-a) > 0$ implies $f(y_{n_k})$ and $f(z_{n_k})$ cannot both converge to the same limit, so $f$ is not continuous at $\eta$.
