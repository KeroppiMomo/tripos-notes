• There is a smallest positive number $\pi$ such that $\cos \pi/2 = 0$. Also $\sin \pi/2 = 1$. (Proof)

  • $\cos z = \sin(z+\pi/2) = -\cos(z+\pi) = \cos(z+2\pi)$ and $\sin z = -\cos(z+\pi/2) = -\sin(z+\pi) = \sin(z+2\pi)$ follow from angle sum formula.

Then

  • $e^{iz+2\pi i} = \cos(z+2\pi) + i\sin(z+2\pi) = \cos z + i \sin z = e^{iz}$ so $e^{w}$ is periodic with period $2\pi i$.

  • $e^{i\pi} = \cos\pi + i \sin \pi = -\cos 0 - i\sin 0 = -1$.
