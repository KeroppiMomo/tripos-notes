We perform bisection to find nested intervals that contain infinitely many elements in the sequence as follows:

  • Step 1: set $a_1=-K$ and $b_1=K$ so that all terms of $(x_n)$ lie in $[a_1, b_1]$.

  • Step $n$: let $c_{n-1} = (a_{n-1} + b_{n-1})/2$. Since there is infinitely many terms in $[a_{n-1}, b_{n-1}]$,

    • there are infinitely many terms in $[a_{n-1}, c_{n-1}]$, in which case set $a_n = a_{n-1}$ and $b_n=c_{n-1}$; or

    • there are infinitely many terms in $[c_{n-1}, b_{n-1}]$, in which case set $a_n = c_{n-1}$ and $b_n = b_{n-1}$.

    Then there are infinitely many terms of $(x_n)$ lying in $[a_n, b_n]$, $b_n-a_n = (b_{n-1} - a_{n-1})/2$, and $a_{n-1}\leq a_n < b_n \leq b_{n-1}$.

By construction, $a_n$ is increasing and bounded above by $b_1$, so $a_n \to a$ for some $a\in[a_1,b_1]$ as $n\to\infty$. Similarly $b_n$ is decreasing and bounded below by $a_1$, so $b_n\to b$ for some $b\in[a_1, b_1]$ as $n\to\infty$.

But $b_n - a_n = (b_{n-1} - a_{n-1})/2$, so by taking limits on both sides, $b-a = (b-a)/2$ so $a=b$.

Now construct the subsequence as follows:

  • Pick $n_1 = 1$.

  • Pick $n_j$ to be the smallest integer larger than $n_{j-1}$ such that $x_{n_j} \in [a_j, b_j]$. This is always possible since $[a_j, b_j]$ contains infinitely many terms in the sequence.

Notice that $a_j \leq x_{n_j} \leq b_j$, so by squeeze theorem, $x_{n_j} \to a$.
