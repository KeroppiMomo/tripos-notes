Sequences open_in_new

Definition (limit of a sequence) open_in_new

For a sequence of real (or complex) numbers $(a_n)_{n=1}^\infty$, we say $a_n\to a$ as $n\to\infty$ for some $a\in\R$ if for all $\epsilon>0$, there exists $N\in\N$ such that $$ |a_n - a| < \epsilon\quad \forall n\geq N. $$

Equivalently you can replace with $|a_n - a| \geq \epsilon$, or $\epsilon$ by any multiple of $\epsilon$.

Axiom (least upper bound) open_in_new

Every non-empty set of real numbers bounded above has a supremum.

Similarly every non-empty set bounded below has a infimum.

Axiom (monotone convergence) open_in_new

An increasing sequence of real numbers which are bounded above converges i.e. has a limit.

Equivalently, a decreasing sequence bounded below converges.

Proof of equivalence with least upper bound

Properties of limits:

  • Limit is unique, i.e. if $a_n \to a$ and $a_n \to b$, then $a=b$. (Proof)

  • Subsequences converge to the same limit, i.e. if $a_n \to a$ and $n_1 < n_2 < n_3 < \ldots \in \N$, then $a_{n_j} \to a$ as $j\to\infty$. (Proof)

  • Operations on sequences: if $a_n \to a$ and $b_n \to b$, then

    • $a_n + b_n \to a+b$; (Proof)

    • $a_n b_n \to ab$; (Proof)

    • if $a_n\neq 0$ and $a\neq 0$, then $1/a_n \to 1/a$; (Proof)

  • Bound of sequence also bounds limit: if $a_n \to a$ and $a_n \leq A$ for all $n\in\N$, then $a\leq A$.

  • Squeeze theorem: if $a_n\to a$, $c_n\to a$ and $a_n \leq b_n \leq c_n$, then $b_n\to a$. (Proof)

Theorem (Bolzano-Weierstrass theorem) open_in_new

Every bounded real sequence has a convergent subsequence.

That is, if $x_n\in\R$ and there exists $K>0$ such that $|x_n|\leq K$ for all $n$, then there exists indices $n_1<n_2<n_3<\ldots$ and $x\in\R$ such that $x_{n_j}\to x$ as $j\to\infty$.

Series open_in_new

Definition (convergence of series) open_in_new

Suppose $a_j\in\R$ or $\C$. We say that $\sum_{j=1}^\infty a_j$ converges to $S\in\R$ if the sequence of partial sums $S_N\to S$ as $N\to\infty$, where $$ S_N = \sum_{j=1}^N a_j. $$ We write $\sum_{j=1}^\infty a_j = S$.

If $S_n$ does not converge, we say that the series diverges.

Properties of series:

  • Linear combination of series: if $\sum_{j=1}^\infty a_j$ and $\sum_{j=1}^\infty b_j$ both converge, then so does $\sum_{j=1}^\infty (\lambda a_j + \mu b_j)$ for any $\lambda, \mu\in\R$, and $\sum_{j=1}^\infty (\lambda a_j + \mu b_j) = \lambda \sum_{j=1}^\infty a_j + \mu \sum_{j=1}^\infty b_j$.

  • Only the tail matters for convergence: if there exists $N\in\N$ such that $a_j = b_j$ for $j\geq N$, then either both series $\fsum j1\infty a_j$ and $\fsum j1\infty b_j$ converge or both diverge.

Lemma (n-th term test) open_in_new

If $\sum a_j$ converges, then $a_j \to 0$ as $j\to\infty$.

We usually use the contrapositive: if $a_j\not\to 0$, then $\sum a_j$ diverges.

Proof

Converse is false: the harmonic series $\fsum j1\infty 1/j$ diverges.

Definition (absolute convergence) open_in_new

Let $a_n \in \C$. If $\fsum n1\infty |a_n|$ converges, then we say $\fsum n1\infty a_n$ is absolutely convergent.

Limit of a function open_in_new

Definition (limit point) open_in_new

Suppose $E\subseteq \C$. $a\in\C$ (not necessarily in $E$) is a limit point of $E$ if for all $\delta>0$, there exists $z\in E$ such that $$ 0 < |z-a| < \delta. $$

Equivalently, if there exists a sequence $(z_n)$ such that $z_n\in E$, $z_n\neq a$, and $z_n \to a$.

Definition (limit of a function) open_in_new

Suppose $E\subseteq\C$, $f: E\to\C$ and $a$ is a limit point of $E$.

We say $f(z) = l$ as $z\to a$ (also denoted as $\lim_{z\to a} f(z) = l$) if for all $\epsilon>0$, there exists $\delta >0$ such that for all $z\in E$ where $0<|z-a|<\delta$, we have $|f(z) - l| < \epsilon$.

Equivalently, if $f(z_n) \to l$ for all sequences $z_n\in E$, $z_n\neq a$ and $z_n \to a$.

Properties of the limit of a function: suppose $E\subseteq \C$, $a\in\C$ is a limit point, $f,g: E\to\C$ and as $z\to a$, $f(z) \to l$ and $g(z) \to k$.

  • The limit is unique, i.e. if $f=g$, then $l=k$.

  • $f(z) + g(z) \to l + k$;

  • $f(z) + g(z) \to lk$;

  • if $k\neq 0$, then $f(z)/g(z) \to l/k$

as $z\to a$.

Lemma (monotone convergence of functions) open_in_new

Suppose $f: [a,\infty) \to \R$ is monotone increasing and bounded above. Then $$ \begin{align*} \lim_{x\to \infty} f(x) = \sup_{x\geq a}f(x). \end{align*} $$

Proof

Definition (continuity) open_in_new

$f: E\to\C$ where $\emptyset\neq E\subseteq\C$ is continuous at a point $a\in E$, equivalently,

  • if for all $\epsilon>0$, there exists $\delta>0$ such that for all $z\in E$ where $|z-a| < \delta$, we have $|f(z) - f(a)| < \epsilon$;

  • if $f(z_n) \to f(a)$ as $n\to\infty$ for all sequences $(z_n)$ of $E$ with $z_n\to a$ (Proof of equivalence);

  • if $f(z) \to f(a)$ as $z\to a$.

$f$ is continuous on $E$ if $f$ is continuous at any $a\in E$.

If $a\in E$ is isolated, i.e. not a limit point, then $f$ is always continuous at $a$.

Properties of continuous functions: if $f,g: E\to\C$ are continuous at a point, then so are

  • $f(z) + g(z)$;

  • $f(z) g(z)$;

  • $\lambda f(z)$ where $\lambda\in\C$;

  • $1/f(z)$, provided that $f(z)\neq 0$ in some neighbourhood of the point.

Using the sequential characterisation of continuity, these follows from the analogous results for sequences.

Differentiability open_in_new

Definition (differentiability) open_in_new

For some $E\subseteq\C$, let $f: E\to\C$ and let $x\in E$ be a limit point.

$f$ is differentiable at $x$, with derivative $f'(x)$, if $$ \lim_{y\to x} \frac{f(y) - f(x)}{y-x} = f'(x). $$

Equivalent definitions, assuming $E$ has no isolated points:

  • Using $h=y-x$: $$ f'(x) = \lim_{h\to0} \frac{f(x+h) - f(x)}{h}. $$

  • Best affine approximation exists: there exists $f'(x)$ and $\epsilon(h)$ such that $$ f(x+h) = f(x) + hf'(x) + \epsilon(h) \quad\text{and}\quad \lim_{h\to0} \frac{\epsilon(h)}{h} = 0. $$

  • Continuous error term: there exists $f'(x)$ and $\epsilon(h)$ such that $$ f(x+h) = f(x) + hf'(x) + h\epsilon(h), $$ with $\epsilon(0)=0$ and continuous at $h=0$. (Derivation of equivalence)

Properties of differentiability open_in_new

Lemma (differentiability implies continuity) open_in_new

If $f: E\to\C$ is differentiable at $x$, then $f$ is continuous at $x$.

Proof

Suppose $f,g$ are differentiable at $x$.

  • $f+g$ is differentiable at $x$ and $(f+g)'(x) = f'(x) + g'(x)$ (Proof).

  • $fg$ is differentiable at $x$ and $(fg)'(x) = f'(x)g(x) + f(x)g'(x)$ (Proof).

  • If $f(y)\neq 0$ for all $y\in E$, then $1/f$ is differentiable at $x$ and $(1/f)' = -f'(x)/f(x)^2$ (Proof).

Lemma (chain rule) open_in_new

Suppose $U,V\subseteq\C$ and $f: U\to V, g: V\to\C$ are such that $f$ is differentiable at $a\in U$ and $g$ is differentiable at $b=f(a)\in V$.

Then $g\circ f: U \to \C$ is differentiable at $a$ and $$ (g\circ f)'(a) = g'\big(f(a)\big)\cdot f'(a) $$

Proof

Theorem (mean value theorem) open_in_new

Let $f: [a,b]\to\R$ be continuous on $[a,b]$ and differentiable on $(a,b)$.

Then there exists $c\in(a,b)$ such that $$ f(b) - f(a) = (b-a)f'(c). $$

Proof

Equivalently, if $f$ is differentiable on an open interval $I$ containing $a$, then for all $h\in\R$ where $a+h\in I$, there exists $\theta\in(0,1)$ such that $$ f(a+h) = f(a) + hf'(a+\theta h). $$

  3. if $f'(x) = 0$ for all $x\in(a,b)$, then $f$ is constant.

Taylor's theorem open_in_new

Definition (higher derivatives) open_in_new

Suppose a function $f: (a,b)\to\R$ is differentiable on $(a,b)$. If $f'(x)$ is a function differentiable at $x\in(a,b)$, then we say $f$ is twice differentiable at $x$ and write $$f''(x) \equiv f^{(2)}(x) = (f')'(x)$$

Iterate to define $k$-times differentiable and $$ f^{(k)}(x) = (f^{(k-1)})'(x). $$

Definition (smooth function) open_in_new

A function $f: (a,b) \to\R$ is smooth if it is $k$-times differentiable on $(a,b)$ for every $k$.

Theorem (Taylor's theorem) open_in_new

Suppose $f$ and its first $n-1$ derivatives are continuous on $[a,a+h]$, and $f$ is $n$-times differentiable on $(a,a+h)$.

Then $$ f(a+h) = P_{n-1}(a+h) + R_n(a+h) $$ where $P_{n-1}$ is the $(n-1)$-th order Taylor polynomial $$ P_{n-1}(a+h) = \fsum i0{n-1} \frac{h^i}{i!} f^{(i)}(a) $$ and the error term $R_n(h)$ can be written in the form of

  • Lagrange remainder: there exists $\theta\in(0,1)$ such that $$ R_n(a+h) = \frac{h^n}{n!}f^{(n)}(a+\theta h); $$

  • Cauchy remainder: there exists $\theta\in(0,1)$ such that $$ R_n(a+h) = \frac{(1-\theta)^{n-1} h^n}{(n-1)!} f^{(n)}(a+\theta h). $$

  • Integral remainder: if we further assume $f^{(n)}$ is continuous on $x\in[a,a+h]$, then $$ \begin{align*} R_n(a+h) = \frac{h^n}{(n-1)!} \int_{0}^{1} (1-t)^{n-1} f^{(n)}(a+th)\, dt. \end{align*} $$

Proof 1 (Lagrange remainder only) Proof 2 (less motivated) Proof (integral remainder)

  • The integral form can be used to recover Cauchy and Lagrange form of the remainder if the additional assumption is made. (Derivation)

  • $f(h)$ being equal to the Taylor series is generally false, even if $f$ is smooth and the series converges.

  • To show a function equals its Taylor series, we need to show $R_n(h)\to 0$ as $n\to\infty$ for fixed $h$, e.g. Proposition (binomial series).

Complex differentiability is much more restrictive than in the reals, e.g. complex conjugate is not differentiable.

But they have much more properties, e.g. suppose $f: D\to\C$ is complex differentiable on $D=\sb{z\in\C}{|z|<1}$. Then

  • $f$ is infinitely differentiable;

  • the Taylor series about 0 of $f$ converges to $f$ on $D$.

Definition (power series) open_in_new

A power series about a point $z_0\in\C$ is a function of the form $$ f(z)= \fsum n0\infty a_n(z-z_0)^n $$ where $z,a_n\in\C$.

Lemma (power series absolutely converges if converges at some outer point) open_in_new

If $\fsum n0\infty a_n z^n$ converges and $|w|<|z|$, then $\fsum n0\infty a_n w^n$ converges absolutely.

Proof

Theorem (radius of convergence) open_in_new

Let $\fsum n0\infty a_n z_n$ be a power series. Then there exists $R\in[0,\infty]$, called the radius of convergence, such that

  • for $|z|<R$, the series converges absolutely,

  • for $|z|>R$, the series diverges.

Proof

  • $R=0$ means that the power series converges only at $z=0$;

  • $R=\infty$ means that the power series converges absolutely for all $z\in\C$.

  • When $0<R<\infty$, we have to consider the convergence on $|z|=R$ on a case by case basis.

Lemma (limit of coefficients determines radius of convergence) open_in_new

If, as $n\to\infty$,

  • $|a_{n+1}/a_n| \to l$; or

  • $|a_n|^{1/n} \to l$,

then the radius of convergence of $\sum a_n z^n$ is $$ R=\frac1l\qquad \text{(take $\tfrac10=\infty, \tfrac1\infty=0$)}. $$

Proof

Theorem (term-by-term differentiation inside radius of convergence) open_in_new

Suppose $f(z)=\fsum n0\infty a_n z^n$ has radius of convergence $R>0$.

Then $f$ is differentiable at all points $z$ with $|z|<R$, and $$ f'(z)=\fsum n1\infty na_n z^{n-1} $$ has a radius of convergence of $R$.

Proof (non-examinable)

Standard functions open_in_new

Definition (exponential function) open_in_new

Define $\exp: \mathbb C\to\mathbb C$ with $$ \exp(z) = \sum_{k=0}^\infty \frac{z^k}{k!} = 1 + z + \frac{1}{2!}z^2 + \frac{1}{3!}z^3 + \ldots. $$

This is well-defined as the series has a radius of convergence of $\infty$, since $|a_{n+1}/a_n| = |k!/(k+1)!| = 1/(k+1) \to 0$.

Properties of exponential function open_in_new

  • $\exp$ is differentiable everywhere and $\exp'(z)=\exp(z)$. (Proof)

  • $\exp(a+b)=\exp(a)\exp(b)$. (Proof)

For $\exp: \R \to \R$,

  • $\exp(x)>0$ for all $x\in\R$. (Proof)

  • $\exp$ is strictly increasing. (Proof)

  • $\exp(x)\to\infty$ as $x\to\infty$, and $\exp(x)\to0$ as $x\to-\infty$. (Proof)

  • $\exp: \R\to(0,\infty)$ is a bijection. (Proof)

Therefore $\exp: (\R,+) \to \big((0,\infty),\times)$ is a group isomorphism.

Definition (logarithm) open_in_new

Let $\log: (0,\infty)\to\R$ be the inverse of $\exp: \R\to(0,\infty)$.

Properties of logarithm open_in_new

  • $\log: (0,\infty)\to\R$ is a bijection, $\log\big(\exp(x)\big) = x$ for all $x\in\R$ and $\exp\big(\log(t)\big) = t$ for all $t\in(0,\infty)$.

  • $\log(st)=\log(s) + \log(t)$. (Proof)

  • $\log(x)\to\infty$ as $x\to\infty$ and $\log(x)\to-\infty$ as $x\to0^+$. (Proof)

  • $l$ is differentiable and monotone, with $l'(t) = 1/t$. (Proof)

Definition (exponentiation) open_in_new

For all $\alpha\in\R$ and $x>0$, define $$ x^\alpha \equiv r_\alpha(x) = \exp(\alpha\log(x)). $$

Properties of exponentiation open_in_new

Suppose $x,y>0$ and $\alpha,\beta\in\R$.

  • $r_\alpha(xy) = r_\alpha(x) r_\alpha(y)$.

  • $r_{\alpha+\beta}(x) = r_\alpha(x) r_\beta(x)$.

  • $r_\alpha(r_\beta(x))=r_{\alpha\beta}(x)$.

  • $r_1(x)=x$ and $r_0(x)=1$.

  • $(x^\alpha)' = \alpha x^{\alpha-1}$ and $(\alpha^x)'=\alpha^x\log\alpha$.

Definition (sin, cos) open_in_new

Define $$\begin{align*} \cos z &= \fsum k0\infty \frac{(-1)^kz^{2k}}{(2k)!} &&= 1-\frac{z^2}{2!} +\frac{z^4}{4!}-\ldots \\ \sin z &= \fsum k0\infty \frac{(-1)^kz^{2k+1}}{(2k+1)!} &&= z-\frac{z^3}{3!} +\frac{z^5}{5!}-\ldots \end{align*}$$ where both series have a radius of convergence $\infty$ by the ratio test.

Properties of trigonometric functions open_in_new

  • $\cos' z = -\sin z$ and $\sin' z = \cos z$ by term-by-term differentiation.

  • $e^{iz} = \cos z + i\sin z$. (Proof)

  • $\cos z = (e^{iz}+e^{-iz})/2$ and $\sin z = (e^{iz}-e^{-iz})/2i$. (Proof)

  • $\cos z = \cos(-z), \sin z = -\sin(-z), \cos 0 = 1, \sin 0 = 0$ by inspection on the series.

  • $\sin(z+w) = \sin z \cos w + \cos z \sin w$ and $\cos(z+w) = \cos z \cos w - \sin z \sin w$. (Proof)

  • $\cos^2 z + \sin^2 z = 1$ by setting $w=-z$.

  • For $x\in\R$, the above gives $|\cos x|,|\sin x|\leq 1$.

Periodicity of trigonometric functions open_in_new

  • There is a smallest positive number $\pi$ such that $\cos \pi/2 = 0$. Also $\sin \pi/2 = 1$. (Proof)

  • $\cos z = \sin(z+\pi/2) = -\cos(z+\pi) = \cos(z+2\pi)$ and $\sin z = -\cos(z+\pi/2) = -\sin(z+\pi) = \sin(z+2\pi)$ follow from angle sum formula.

Then

  • $e^{iz+2\pi i} = \cos(z+2\pi) + i\sin(z+2\pi) = \cos z + i \sin z = e^{iz}$ so $e^{w}$ is periodic with period $2\pi i$.

  • $e^{i\pi} = \cos\pi + i \sin \pi = -\cos 0 - i\sin 0 = -1$.

Definition (sinh, cosh) open_in_new

For all $z\in\C$, $$ \begin{align*} \cosh z &= \frac{1}{2} (e^{z} + e^{-z}), \\ \sinh z &= \frac{1}{2} (e^{z} - e^{-z}). \end{align*} $$

Properties of hyperbolic functions open_in_new

  • $\cosh' z = \sinh z$ and $\sinh' z = \cosh z$.

  • $\cosh z = \cos iz$ and $\sinh z = -i\sin iz$.

  • $\cosh^2 z - \sinh^2 z = 1$.

Definition (dissection) open_in_new

A dissection $\mathcal D$ of $[a,b]$ is a finite subset of $[a,b]$ containing the endpoints $a,b$.

We write a dissection as $\set{x_0, x_1, \ldots, x_n}$ where $a=x_0 < x_1 < \ldots < x_n = b$.

Definition (Riemann sum) open_in_new

For a bounded function $f: [a,b] \to \R$ and a dissection $\mathcal D$, the associated Riemann upper and lower sums are $$ \begin{align*} U(f,\mathcal D) &= \sum_{j=1}^{n} (x_{j} - x_{j-1}) \sup_{x\in[x_{j-1},x_{j}]} f(x), \\ L(f,\mathcal D) &= \sum_{j=1}^{n} (x_{j} - x_{j-1}) \inf_{x\in[x_{j-1},x_{j}]} f(x). \end{align*} $$

Lemma (inequality of lower and upper Riemann sums) open_in_new

Suppose $\mathcal D_1$ and $\mathcal D_2$ are two dissections of $[a,b]$ and $f:[a,b]\to\R$ is bounded.

Then $L(f,\mathcal D_1) \leq U(f, \mathcal D_2)$.

Proof

Definition (lower and upper integrals) open_in_new

Elementary properties of the integral open_in_new

Let $f,g$ be bounded and integrable on $[a,b]$.

  • If $f(x) \leq g(x)$ for all $x\in[a,b]$, then $\int_{a}^{b} f \leq \int_{a}^{b} g$. (Proof)

  • If $\lambda\in\R$ then $\lambda f$ is also integrable and $\int_{a}^{b} \lambda f = \lambda \int_{a}^{b} f$. (Proof)

  • $f+g$ is integrable with $\int_{a}^{b} f+g = \int_{a}^{b} f + \int_{a}^{b} g$. (Proof)

  • $|f|$ is integrable with $|\int_{a}^{b} f| \leq \int_{a}^{b} |f|$. (Proof)

  • $fg$ is integrable. (Proof)

Additivity:

Theorem (fundamental theorem of calculus 1) open_in_new

Consider a bounded and integrable function $f:[a,b] \to \R$. Let $$ \begin{align*} F(x) = \int_{a}^{x} f(t)\, dt, \quad x\in[a,b]. \end{align*} $$ $F$ is continuous. (Proof)

If $f$ is continuous at $x$, then $F$ is differentiable at $x$ with $F'(x) = f(x)$. (Proof)

Theorem (fundamental theorem of calculus 2) open_in_new

Suppose $G:[a,b]\to\R$ is continuous on $[a,b]$ and differentiable on $(a,b)$.

If $G'$ is integrable, then $$ \begin{align*} \int_{a}^{b} G'(x)\, dx = G(b) - G(a). \end{align*} $$

Proof

FToC 1 can be used to prove a weaker version of FToC 2 where $G'$ is assumed to be continuous. (Proof)

All continuous functions have an indefinite integral or anti-derivative $$ \begin{align*} \int f(x) \, dx = \int_{a}^{x} f(t)\, dt \end{align*} $$ where $a$ is arbitrary, which is defined up to a constant.

Summary: $$ \begin{align*} \text{FTC 1}: && \dd{x} \int_{a}^{x} f(t)\, dt &= f(x) &&\text{if $f$ is continuous at $x$.} \\ \text{FTC 2}: && \int_{a}^{b} \dd{x}[f(x)]\, dx &= f(b)-f(a) &&\text{if $f'$ is integrable.} \end{align*} $$

Improper integrals open_in_new

Lebesgue integral is the full treatment for functions whose domain or image is unbounded, but now we just use improper Riemann integrals.

Definition (integral on infinite domain) open_in_new

  • Suppose $f:[a,\infty]\to\R$ is integrable on each interval $[a,R]$ where $R>a$. If the following limit exists, we say $\int_{a}^{\infty} f(x)\, dx$ exists or converges and equals $$ \begin{align*} \int_{a}^{\infty} f(x)\, dx \equiv \lim_{R\to \infty} \int_{a}^{R} f(x)\, dx. \end{align*} $$

    If there is no limit, we say $\int_{a}^{\infty} f(x)\, dx$ diverges.

  • Similar for $\int_{-\infty}^{a} f(x)\, dx$.

  • If both of the following limits exist, then $\int_{-\infty}^{\infty} f(x)\, dx$ exists and equals $$ \begin{align*} \int_{-\infty}^{\infty} f(x)\, dx \equiv \int_{-\infty}^{a} f(x)\, dx + \int_{a}^{\infty} f(x)\, dx \end{align*} $$ where $a$ is arbitrary.

Caution: $\int_{-\infty}^{\infty} f(x)\,dx \not\equiv \lim_{R\to \infty} \int_{-R}^{R} f(x)\, dx$.

Definition (integral with an isolated singularity) open_in_new

  • Suppose $f:(b,c]\to\R$ is integrable on each $[b+\delta, c]$ for $0<\delta<c-b$. If the following limit exists, then we say $\int_{b}^{c} f(x)\, dx$ exists or converges and equals $$ \begin{align*} \int_{b}^{c} f(x)\, dx \equiv \lim_{\delta\to 0^+} \int_{b+\delta}^{c} f(x)\, dx. \end{align*} $$

  • Similar for $f:[a,b)\to\R$.

  • If $f:[a,c]\setminus\set{b}\to\R$ and the following RHS integrals exist, we say $\int_{a}^{c} f(x)\, dx$ exists and equals $$ \begin{align*} \int_{a}^{c} f(x)\, dx \equiv \int_{a}^{b} f(x)\, dx + \int_{b}^{c} f(x)\, dx. \end{align*} $$

Definition (measure zero) open_in_new

A subset $A\subset \R$ has measure zero if for all $\epsilon>0$, there exists a countable collection of intervals $I_j$ such that $$ \begin{align*} A\subset \bigcup_{j=1}^\infty I_j \quad \text{and}\quad \sum_{j=1}^{\infty} |I_j| < \epsilon \end{align*} $$ where $|I_j|$ denotes the length of $I_j$.

  • If $B$ has measure zero and $A\subset B$, then $A$ also has measure zero.

  • If $A_k$ has measure zero for each $k\in\N$, then $\bigcup_{k\in\N} A_k$ has measure zero. (Proof) This implies countable sets have measure zero.

Definition (oscillation of a function) open_in_new

  • For an interval $I$, the oscillation of a bounded function $f$ defined on $I$ is $$ \begin{align*} \omega_f(I) \equiv \sup_{I} f - \inf_I f. \end{align*} $$

  • At a point $x$, the oscillation of $f$ is $$ \begin{align*} \omega_f(x) \equiv \lim_{\epsilon\to 0} \omega_f((x-\epsilon, x+\epsilon)). \end{align*} $$

If $I'\subset I$, then $\omega_f(I') \leq \omega_f(I)$. Also $\omega_f$ is bounded below by 0, so the limit exists.
Lemma (continuous iff zero oscillation) open_in_new

$f$ is continuous at $x$ iff $\omega_f(x) = 0$.
