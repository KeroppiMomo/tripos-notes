Let $f: [a,b]\to\R$ be a continuous function which is differentiable on $(a,b)$. Then

  1. if $f'(x)>0$ for all $x\in(a,b)$, then $f$ is strictly increasing;

  2. if $f'(x)\geq 0$ for all $x\in(a,b)$, then $f$ is increasing;

  3. if $f'(x) = 0$ for all $x\in(a,b)$, then $f$ is constant.
