Let $S_n = \fsum k1n (-1)^{k+1} a_k$.

Since $(a_n)$ is decreasing, we have $$ \begin{align*} S_{2n} &= S_{2n-2} + a_{2n-1} - a_{2n} \geq S_{2n-2}, \\ S_{2n+1} &= S_{2n-1} - a_{2n} + a_{2n+1} \leq S_{2n-1}. \end{align*} $$ Moreover $a_n \geq 0$, so $$ S_{2n+1} = S_{2n} + a_{2n-1} \geq S_{2n}. $$ Therefore we have inequalities $$ S_2 \leq S_4 \leq \ldots \leq S_{2n} \leq S_{2n+1} \leq \ldots \leq S_3 \leq S_1. $$

Thus $(S_{2n})_{n=1}^\infty$ is increasing and bounded above by $S_1$, so $S_{2n}$ converges to some limit $S$. And $(S_{2n+1})_{n=1}^\infty$ is decreasing and bounded below by $S_2$, so $S_{2n+1}$ converges to some limit $S'$.

But $$ S_{2n+1} = S_{2n} + a_{2n-1} \to S+0 = S $$ so $S' = S$.

Hence given any $\epsilon>0$, there exists $N_1, N_2$ such that $|S_{2n}-S| < \epsilon$ if $n\geq N_1$ and $|S_{2n+1} - S| < \epsilon$ if $n\geq N_2$. Therefore for all $n\geq\max\set{2N_1, 2N_2+1}$, $|S_n - S| < \epsilon$. So $S_n \to S$.
