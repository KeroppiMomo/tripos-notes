We know that for all $\epsilon>0$, there exists $N_1(\epsilon)$ and $N_2(\epsilon)$ such that $|a_n - a| < \epsilon$ if $n\geq N_1(\epsilon)$ and $|a_n - b| < \epsilon$ if $n\geq N_2(\epsilon)$.

So for $n\geq\max\{N_1(\epsilon), N_2(\epsilon)\}$, $$\begin{align*} 0 \leq |b-a| &= |b-a_n + a_n - a| \\ &\leq |b-a_n| + |a_n - a| \\ &< 2\epsilon. \end{align*}$$ If $a\neq b$, then picking $\epsilon = |b-a|/3$ yields a contradiction. Hence $a=b$.
