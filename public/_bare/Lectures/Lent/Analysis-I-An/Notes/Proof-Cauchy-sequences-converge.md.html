Suppose $(a_n)_{n\in\N}$ is Cauchy.

$(a_n)_{n\in\N}$ is bounded

There exists $N\in\N$ such that $|a_n - a_m| < 1$ for all $n,m\geq N$. So for all $n\geq N$, $$ |a_n| = |a_n - a_N + a_N| \leq |a_n - a_N| + |a_N| < 1+|a_N|, $$ which is some constant.

Thus for all $n$, $$|a_n| \leq 1+\max\set{|a_1|, |a_2|, \ldots, |a_N|}.$$

Hence by Bolzano-Weierstrass theorem, there exists $a\in\R$ and $n_1<n_2<\ldots$ such that $$ a_{n_j} \to a \quad\text{as } j\to\infty. $$

Limit of subsequence is limit of sequence

Note that for any $n,j\in N$, $$ |a_n - a| = |a_n - a_{n_j} + a_{n_j} - a| \leq |a_n - a_{n_j}| + |a_{n_j} - a|. $$

For any $\epsilon>0$, since $(a_n)$ is Cauchy, let $N_1$ such that $|a_n-a_m|<\epsilon$ for all $n,m\geq N_1$. Since $a$ is the limit of the subsequence, let $N_2$ such that $|a_n-a_m| < \epsilon$.

Then by picking some $j\geq \max\set{N_1, N_2}$, we have for all $n\geq N_1$, $$ |a_n - a| \leq \epsilon+\epsilon = 2\epsilon $$ so $a_n\to a$ as $n\to\infty$.
