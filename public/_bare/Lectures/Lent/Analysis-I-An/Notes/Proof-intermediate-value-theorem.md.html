Let $S=\sb{x\in[a,b]}{f(x) < \eta}$.

$S$ is non-empty since $a\in S$. Also $S$ is bounded above by $b$. Hence $S$ has a supremum $c$.

From the definition of a supremum, for all $n\in\N$, there exists $x_n\in S$ with $$ c - \frac1n \leq x_n \leq c. $$ Therefore $x_n\to c$ and since $f$ is continuous, we have $f(x_n) \to f(c)$. Since $f(x_n) < \eta$, we obtain $f(c)\leq \eta$.

In particular, $c\neq b$. Then we can let $y_n = c+1/n$ and for $n$ large enough, $y_n\in[a,b]$. Note that $y_n\to c$. Furthermore since $y_n>c$, $y_n\not\in S$ so $f(y_n) \geq \eta$. By continuity, $f(y_n) \to f(c)$ so $f(c) \geq \eta$.

Combining both inequalities yields $f(c) = \eta$.
