Suppose $p,q\in\N$. Here we use $r_\alpha(x)$ to refer to the general real exponentiation and $x^\alpha$ to refer to rational exponentiation.

  • Natural number exponents: $r_p(x) = r_{\ub{1+\ldots+1}_{p\text{ times}}}(x) = \ub{r_1(x)\cdot\ldots r_1(x)}_{p\text{ times}} = \ub{x\cdot\ldots\cdot x}_{p\text{ times}} = x^p$.

  • Integer exponents: $r_p(x)\cdot r_{-p}=r_0(x) = 1$ so $r_{-p}(x) = 1/x^p$.

  • Reciprocal exponents: $(r_{1/q}(x))^q = \ub{r_{1/q}(x)\cdot\ldots\cdot r_{1/q}(x)}_{q\text{ times}} = r_{\ub{1/q+\ldots+1/q}_{q\text{ times}}}(x) = r_1(x) = x$ so $r_{1/q}(x) = x^{1/q}$.

  • Rational exponents: $r_{p/q}(x) = r_{\ub{1/q+\ldots+1/q}_{p\text{ times}}}(x) = \big(r_{1/q}(x)\big)^p = (x^{1/q})^p = x^{p/q}$.
