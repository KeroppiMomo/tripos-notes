For any $\epsilon>0$, let $N_1(\epsilon)$ such that $|a_n - a| < \epsilon$ for $n\geq N_1(\epsilon)$ and let $N_2(\epsilon)$ such that $|b_n - b| < \epsilon$ for $n\geq N_2(\epsilon)$.

Now $$\begin{align*} |a_nb_n - ab| &= |a_nb_n - a_nb + a_nb - ab| \\ &\leq |a_n||b_n - b| + |b||a_n - a|. \end{align*}$$

Note that if $n\geq N_1(1)$, then $$ |a_n| = |a_n - a + a| \leq |a_n - a| + |a| < 1 + |a|. $$

Hence if $n\geq\max\{N_1(\epsilon), N_2(\epsilon)\}$, $$\begin{align*} |a_nb_n - ab| &< (1+|a|)\epsilon + |b|\epsilon \\ &= (1 + |a| + |b|)\epsilon \end{align*}$$ which can be made arbitrarily small.
