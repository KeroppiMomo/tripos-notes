Show that if $|x|<1$, then $x^n\to 0$ as $n\to\infty$.

First consider when $0<x<1$. Then $(x^n)_{n\in\N}$ is a decreasing sequence bounded below by 0, so it has a limit $l$.

$x^{n+1} = x\cdot x^n \to xl$ but also it is a subsequence so $x^{n+1} \to l$. Hence by uniqueness of limits, $xl=l$ so $l=0$ as $x\neq 1$.

Now for $-1<x<1$, note that $$ -|x|^n \leq x^n \leq |x|^n. $$ Since $-|x|^n\to 0$ and $|x|^n\to 0$, $x^n\to 0$ by the squeeze theorem.
