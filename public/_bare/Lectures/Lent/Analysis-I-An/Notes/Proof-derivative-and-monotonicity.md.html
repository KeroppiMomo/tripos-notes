1. Let $a\leq x<y\leq b$. By the mean value theorem, there exists $c\in(a,b)$ such that $$f(y) - f(x) = f'(c)\,(y-x) > 0.$$ Hence $f(y) > f(x)$.

  2. Same as above but use $f'(c) \geq 0$.

  3. For all $x\in(a,b]$, apply the mean value theorem on $[a,x]$ to deduce there exists $c\in(a,x)$ such that $$f(x)-f(a) = f'(c)\,(x-a) = 0$$ so $f(x) = f(a)$.

Let $f: \C\to\C$ with $f'(z)=0$. Consider $g: \R\to\C$ where $g(t)=f(tz)$ for some fixed $z\in\C$. By chain rule, $g'(t)=zf'(tz) = 0$.

We can write $g(t)=u(t)+iv(t)$ where $u,v\in\R$. Then $0 = g'(t)=u'(t)+iv'(t)$, so $u'(t) = v'(t) = 0$. By 1., $u,v$ are constant, so $F(z)=F(0)$ for all $z$.
