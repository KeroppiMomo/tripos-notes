Definition (measure zero) open_in_new

A subset $A\subset \R$ has measure zero if for all $\epsilon>0$, there exists a countable collection of intervals $I_j$ such that $$ \begin{align*} A\subset \bigcup_{j=1}^\infty I_j \quad \text{and}\quad \sum_{j=1}^{\infty} |I_j| < \epsilon \end{align*} $$ where $|I_j|$ denotes the length of $I_j$.

  • If $B$ has measure zero and $A\subset B$, then $A$ also has measure zero.

  • If $A_k$ has measure zero for each $k\in\N$, then $\bigcup_{k\in\N} A_k$ has measure zero. (Proof) This implies countable sets have measure zero.

Definition (oscillation of a function) open_in_new

  • For an interval $I$, the oscillation of a bounded function $f$ defined on $I$ is $$ \begin{align*} \omega_f(I) \equiv \sup_{I} f - \inf_I f. \end{align*} $$

  • At a point $x$, the oscillation of $f$ is $$ \begin{align*} \omega_f(x) \equiv \lim_{\epsilon\to 0} \omega_f((x-\epsilon, x+\epsilon)). \end{align*} $$

If $I'\subset I$, then $\omega_f(I') \leq \omega_f(I)$. Also $\omega_f$ is bounded below by 0, so the limit exists.
Lemma (continuous iff zero oscillation) open_in_new

$f$ is continuous at $x$ iff $\omega_f(x) = 0$.

Theorem (Lebesgue's criterion for Riemann integrability) open_in_new

If $f:[a,b]\to\R$ is bounded, then $f$ is Riemann integrable iff $D = \sb{x\in[a,b]}{f \text{ discontinuous at }x}$ has measure zero.
