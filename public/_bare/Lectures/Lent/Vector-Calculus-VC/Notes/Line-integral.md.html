Definition (line integral of a scalar field) open_in_new

For a scalar field $\phi: \R^n\to\R$, the line integral of $\phi(\v x)$ along a curve $C: \v x(t)$ from $\v x(t_a) = \v a$ to $\v x(t_b) = \v b$ where $t_a < t_b$ is $$ \int_C\phi\,ds = \int_{t_a}^{t_b} \phi\big(\v x(t)\big)\,|\dv x|\,dt. $$

In particular for parametrisation using arc length, $$\int_C\phi\, ds = \int_{s_a}^{s_b} \phi\big(\v x(s)\big)\,ds. $$

Definition (line integral of a vector field) open_in_new

For a vector field $\v F: \R^n \to \R^n$, the line integral of $\v F(\v x)$ along the curve $C:\v x(t)$ from $\v x(t_a) = \v a$ to $\v x(t_b) = \v b$ is $$ \int_C \v F\cdot d\v x = \int_{t_a}^{t_b} \v F\big(\v x(t)\big)\cdot \dv x(t)\, dt. $$

When $C$ is a closed path, i.e. $\v a = \v b$, the integral is called the circulation of $\v F$ around $C$, denoted $$ \oint_C \v F\cdot d\v x. $$

Properties of line integrals open_in_new

  • The line integral of a vector field is the integral of $\v F$ tangent to the curve.

  • Both line integrals are invariant under reparametrisation.

  • The line integral of a scalar field does not depend on orientation since we impose $t_a<t_b$: $$\int_{-C} \phi\,ds = \int_C \phi\,ds,$$ but the line integral of a vector field does: $$\int_{-C} \v F\cdot d\v x = -\int_C \v F\cdot d\v x.$$

  • For a piecewise smooth curve $C_1 + C_2$, $$\int_{C_1+C_2}\v F\cdot d\v x = \int_{C_1}\v F\cdot d\v x + \int_{C_2}\v F\cdot d\v x.$$

Definition (partial differentiation) open_in_new

For a scalar field $\phi: \R^n \to \R$, the partial derivatives are defined to be $$ \pdfd\phi{x^1}(\v x) = \lim_{\epsilon\to0}\frac1\epsilon\big(\phi(x^1 + \epsilon, x^2, x^3, \ldots, x^n) - \phi(x^1, x^2, \ldots, x^n)\big)$$ and similarly for $\partial\phi/\partial x^i$.

Definition (gradient) open_in_new

For a scalar field $\phi: \R^n \to \R$, let $\set{\v e_i}$ be an orthonormal basis of $\R^n$. The gradient of $\phi$ is a vector field defined as $$ \v\nabla\phi = \pdfd\phi{x^i} \v e_i. $$

Definition (conservative vector field) open_in_new

A vector field $\v F$ is called conservative if it can be written as $$ \v F = \v\nabla \phi $$ for some scalar field $\phi$ called the potential.

Condition for a conservative vector field open_in_new

For a vector field $\v F$, a necessary condition is $$ \pdfd{F_i}{x^j} = \frac{\partial^2\phi}{\partial x^i \partial x^j} = \pdfd{F_j}{x^i}.$$ This turns out to be a sufficient condition for well-behaved functions.

Definition (exact differential) open_in_new

Given a function $\phi(\v x)$, its exact differential is $$ d\phi = \pdfd \phi{x^i} dx^i = \nabla\phi \cdot d\v x.$$

For a vector field $\v F$, $\v F\cdot d\v x$ is an exact differential form if it can be written as $\v F\cdot d\v x=d\phi$. This means $\v F=\nabla\phi$.

If a vector field has singular points, even if it is conservative, $\oint_C \v F\cdot d\v x$ may not be zero for a curve $C$ around the singular point.
