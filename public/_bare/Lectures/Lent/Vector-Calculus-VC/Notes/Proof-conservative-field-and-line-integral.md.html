2 => 1: closed curves have the same endpoints so the integral is the same as a curve that does not move.

1 => 2: suppose curves $C_1$ and $C_2$ have the same endpoints. Then the curve $C_1 - C_2$ is closed. Hence $$ 0 = \oint_{C_1 - C_2}\v F\cdot d\v x = \int_{C_1}\v F\cdot d\v x - \int_{C_2}\v F\cdot d\v x $$ as required.

3 => 2: let $\v F=\v\nabla\phi$. Then along any open curve $C$ parametrised by $\v x(t)$, $$\begin{align*} \int_C \v F\cdot d\v x &= \int_C \nabla\phi\cdot d\v x \\ &= \int_{t_a}^{t_b} \pdfd\phi{x^i} \dfd{x^i}t dt \\ &= \int_{t_a}^{t_b} \dfd{\phi\big(\v x(t)\big)} t \,dt \\ &= \phi\big(\v x(t_b)\big) - \phi\big(\v x(t_a)\big) \end{align*}$$ which only depends on endpoints.

2 => 3: suppose $\int_C\v F\cdot d\v x$ only depends on endpoints. We need to construct $\phi$.

Set $\phi(\v 0) = 0$, and define $$ \phi(\v y) = \int_{C(\v y)} \v F\cdot d\v x $$ where $C(\v y)$ is any path from $\v 0$ to $\v y$. Any path gives the same integral by hypothesis.

Now we check that $\v F=\nabla\phi$: $$\begin{align*} \pdfd\phi{x^i}(\v y) &= \lim_{\epsilon\to0} \frac1\epsilon\bigg[\int_{C(\v y + \epsilon\v e_i)} \v F\cdot d\v x - \int_{C(\v y)} \v F\cdot d\v x\bigg] \\ &= \lim_{\epsilon\to 0} \frac1\epsilon\int_\v y^{\v y + \epsilon\v e_i} \v F\cdot d\v x \\ &= \lim_{\epsilon\to 0} \frac1\epsilon\big(\epsilon F_i(\v y) + O(\epsilon^2)\big) \\ &= F_i(\v y) \end{align*}$$ so $$\nabla\phi = \v F.$$
