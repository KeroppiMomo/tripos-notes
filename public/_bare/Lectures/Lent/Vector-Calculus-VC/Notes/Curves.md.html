Definition (parametrised curve) open_in_new

For a continuous function $$ f: \R \to \R^n, $$ the image of $f$ is called a parametrised curve with $t$ the parameter.

The choice of parametrisation is not unique, and plotting the curve loses information about the parameter $t$.

Differentiating a curve open_in_new

Definition (differentiation of vector function) open_in_new

A function $\v x(t)$ is differentiable at $t$ if $$\v x(t+\delta t) - \v x(t) = \dv x(t) \delta t + O(\delta t^2). $$ Then the derivative is $$ \dfd{\v x}t \equiv \dv x = \lim_{\delta t\to0}\frac{\delta\v x}{\delta t}$$ where $\delta\v x(t) = \v x(t+\delta t) - \v x(t)$.

Definition (smooth curve) open_in_new

If $f: \R \to \R^n$ is differentiable everywhere, the curve is said to be smooth.

Properties of differentiation of vector function open_in_new

  • In any bases independent of $t$, we just differentiate vectors component-wise: $$ \v x(t) = x^i(t) \v e_i \implies \dv x(t) = \dot x^i(t) \v e_i. $$

For real function $f(t)$ and vector functions $\v g(t), \v h(t)$, these can be seen by applying the product rules to the components:

  • $\dd t(f\v g) = \dot f\v g + f\dv g$;

  • $\dd t (\v g\cdot\v h) = \dv g\cdot\v h + \v g\cdot\dv h$;

  • $\dd t (\v g\times\v h) = \dv g\times \v h + \v g\times\dv h$.

Tangent vector open_in_new

Definition (tangent vector) open_in_new

The derivative $\dv x(t)$ is called a tangent vector to the curve at $t$.

The direction of $\dv x(t)$ is independent of the choice of parametrisation up to a sign, but the magnitude is not.

Definition (regular parametrisation) open_in_new

A parametrisation is called regular if $\dv x(t) \neq \v 0$ for any $t$.

We assume regularity from now on.

Arc length open_in_new

Definition (arc length of a curve) open_in_new

For some starting point $t_0$, the arc length $s(t)$ is the (signed) distance from $t_0$ to $t$, $$ s(t) = \int_{t_0}^t |\dv x(t')|\,dt'. $$

  • The arc length between two points on a curve is independent of the choice of parametrisation. (Proof)

  • Using $s$ to parametrise the curve i.e. $\v x(s)$, the tangent vector has unit norm: $$ \bigg|\dvd xs\bigg| = \frac{|d\v x/dt|}{ds/dt} = 1. $$

Curvature open_in_new

Definition (curvature of a curve) open_in_new

Let $\v x(s)$ be a parametrisation by the arc length. The curvature is $$ \kappa(s) = \bigg|\dvd[2]xs\bigg| = \bigg|\dvd ts\bigg| $$ where $\v t = d\v x/ds$ is the unit tangent vector.

If the curve is locally a circle of radius $R$, the curvature at that point is $1/R$.

Definition (principal normal) open_in_new

The principal normal at a point $\v x(s)$ is $$ \v n = \frac1\kappa\dvd[2]xs = \frac1\kappa\dvd ts. $$ $1/\kappa$ ensures this is a unit vector.

Proposition (principal normal perpendicular to tangent) open_in_new

If curvature $\kappa\neq 0$, then $\v n\cdot\v t=0$.

Proof

Defintion (osculating plane) open_in_new

The plane spanned by $\v t$ and $\v n$ is called the osculating plane.

  • In $\R^3$, since $\v t\cdot\dvd ts=0$, we can compute the curvature as $$ \kappa = \bigg|\v t\times\dvd ts\bigg| $$

Binormal open_in_new

Definition (binormal) open_in_new

For curves in $\R^3$, define the binormal as $$ \v b = \v t\times\v n. $$

When $\kappa(s)\neq 0$, the vectors $\v t, \v n, \v b$ form an orthonormal basis for each $s$.

Definition (torsion) open_in_new

For a curve in $\R^3$, the torsion $\tau(s)$ is defined as $$ \dvd bs = -\tau\v n. $$

Derivation that change in b is parallel to n

The torsion measure how much the curve twists out of the plane.

Curvature and torsion specify a curve open_in_new

The tangent vectors and binormal vectors satisfy two first order differential equations (so in total six ODEs accounting the components): $$\begin{cases} \dvd ts = \kappa\v n = \kappa\v b\times\v t \\ \dvd bs = -\tau\v n = \tau\v t\times\v b \end{cases}$$ So for fixed $\kappa(s)$ and $\tau(s)$, there is a unique solution given some initial $\v b(0)$ and $\v t(0)$.

Hence the curvature and torsion function specifies a curve up to rotation and translation.

Line integral open_in_new

Manipulating curves open_in_new

  • The choice of direction along $C$ is called an orientation.

  • The curve $-C$ is the curve $C$ with the opposite orientation.

  • For curves $C_1, C_2$, we can piece together the curves to get $C=C_1 + C_2$.

Definition (line integral of a scalar field) open_in_new

For a scalar field $\phi: \R^n\to\R$, the line integral of $\phi(\v x)$ along a curve $C: \v x(t)$ from $\v x(t_a) = \v a$ to $\v x(t_b) = \v b$ where $t_a < t_b$ is $$ \int_C\phi\,ds = \int_{t_a}^{t_b} \phi\big(\v x(t)\big)\,|\dv x|\,dt. $$

In particular for parametrisation using arc length, $$\int_C\phi\, ds = \int_{s_a}^{s_b} \phi\big(\v x(s)\big)\,ds. $$

Definition (line integral of a vector field) open_in_new

For a vector field $\v F: \R^n \to \R^n$, the line integral of $\v F(\v x)$ along the curve $C:\v x(t)$ from $\v x(t_a) = \v a$ to $\v x(t_b) = \v b$ is $$ \int_C \v F\cdot d\v x = \int_{t_a}^{t_b} \v F\big(\v x(t)\big)\cdot \dv x(t)\, dt. $$

When $C$ is a closed path, i.e. $\v a = \v b$, the integral is called the circulation of $\v F$ around $C$, denoted $$ \oint_C \v F\cdot d\v x. $$

Properties of line integrals open_in_new

  • The line integral of a vector field is the integral of $\v F$ tangent to the curve.

  • Both line integrals are invariant under reparametrisation.

  • The line integral of a scalar field does not depend on orientation since we impose $t_a<t_b$: $$\int_{-C} \phi\,ds = \int_C \phi\,ds,$$ but the line integral of a vector field does: $$\int_{-C} \v F\cdot d\v x = -\int_C \v F\cdot d\v x.$$

  • For a piecewise smooth curve $C_1 + C_2$, $$\int_{C_1+C_2}\v F\cdot d\v x = \int_{C_1}\v F\cdot d\v x + \int_{C_2}\v F\cdot d\v x.$$

Definition (partial differentiation) open_in_new

For a scalar field $\phi: \R^n \to \R$, the partial derivatives are defined to be $$ \pdfd\phi{x^1}(\v x) = \lim_{\epsilon\to0}\frac1\epsilon\big(\phi(x^1 + \epsilon, x^2, x^3, \ldots, x^n) - \phi(x^1, x^2, \ldots, x^n)\big)$$ and similarly for $\partial\phi/\partial x^i$.

Definition (gradient) open_in_new

For a scalar field $\phi: \R^n \to \R$, let $\set{\v e_i}$ be an orthonormal basis of $\R^n$. The gradient of $\phi$ is a vector field defined as $$ \v\nabla\phi = \pdfd\phi{x^i} \v e_i. $$

Definition (conservative vector field) open_in_new

A vector field $\v F$ is called conservative if it can be written as $$ \v F = \v\nabla \phi $$ for some scalar field $\phi$ called the potential.

Theorem (conservative field and line integral) open_in_new

The following are equivalent: for a vector field $\v F: \R^n \to \R^n$,

  1. $\oint_C \v F\cdot d\v x = 0$ for all closed curve $C$;

  2. $\int_C \v F\cdot d\v x$ only depends the endpoints of $C$.

  3. $\v F$ is conservative;

Proof

Condition for a conservative vector field open_in_new

For a vector field $\v F$, a necessary condition is $$ \pdfd{F_i}{x^j} = \frac{\partial^2\phi}{\partial x^i \partial x^j} = \pdfd{F_j}{x^i}.$$ This turns out to be a sufficient condition for well-behaved functions.

Definition (exact differential) open_in_new

Given a function $\phi(\v x)$, its exact differential is $$ d\phi = \pdfd \phi{x^i} dx^i = \nabla\phi \cdot d\v x.$$

For a vector field $\v F$, $\v F\cdot d\v x$ is an exact differential form if it can be written as $\v F\cdot d\v x=d\phi$. This means $\v F=\nabla\phi$.

If a vector field has singular points, even if it is conservative, $\oint_C \v F\cdot d\v x$ may not be zero for a curve $C$ around the singular point.
