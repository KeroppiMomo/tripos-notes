Proof (only isotropic tensors with rank under 3)

  • For an isotropic rank 1 tensor, we have $T_i = T_i' = R_{ij} T_j$.

    Pick $R_{ij} = -\delta_{ij}$ to see $T_i = -T_i$, so $T_i = 0$.

  • For an isotropic rank 2 tensor, we have $T_{ij} = T_{ij}' = R_{ik} R_{jl} T_{kl}$.

    Pick the $\pi/2$ rotation matrix $$ \begin{align*} R_{ij} = \pmat{0 & 1 & 0 \\ -1 & 0 & 0 \\ 0 & 0 & 1} \end{align*} $$ to see $T_{13}' = R_{12} R_{33} T_{23} = T_{23}$ and $T_{23}' = R_{21} R_{33} T_{13} = -T_{13}$, so $T_{13}=T_{23}=0$. Also $T_{11}' = R_{12}R_{12}T_{22} = T_{22}$ so $T_{11} = T_{22}$.

    Similar arguments show that $T_{ij}=0$ for $i\neq j$ and $T_{11} = T_{22} = T_{33} = \alpha$ for some $\alpha$.

  • For an isotropic rank 3 tensor, we have $T_{ijk} = T_{ijk}' = R_{il} R_{jp} R_{kq} T_{lpq}$. (We insist $R\in\mathrm{SO}(3)$ to allow $\epsilon_{ijk}$.)

    Pick the $\pi/2$ rotation matrix $$ \begin{align*} R_{ij} = \pmat{0 & 1 & 0 \\ -1 & 0 & 0 \\ 0 & 0 & 1} \end{align*} $$ to see $T_{123} = -T_{213}$. Similarly swapping any two indices in $T_{123}$ gives a negative sign.

    Now pick the $\pi$ rotation matrix $$ \begin{align*} R_{ij} = \pmat{-1 & 0 & 0 \\ 0 & -1 & 0 \\ 0 & 0 & 1} \end{align*} $$ to see $T_{133} = -T_{133}$, $T_{111} = -T_{111}$ etc, so $T_{ijk}$ unless $i,j,k$ are distinct.

    Therefore $T_{ijk} = \beta \epsilon_{ijk}$.
